# TypeScript BoilerPlate

## Description

this is a boilerplate for typescript projects

- SWC
- ESlint
- Prettier
- Jest

## Installation

```bash
$ git clone
$ yarn install
```

## Usage

- customize package.json scripts for your project

### Start

```bash
$ yarn start
```

### Build

```bash
# remove build output
$ yarn prebuild

$ yarn build
```

### Test

```bash
$ yarn test

# Test with coverage report
$ yarn test:cov
```

### lint

```bash
$ yarn lint
$ yarn lint:fix
```
